# CrateDig Brand Assets
## Direction: Dark Vinyl — Record Store Noir

## Quick Start
1. Read **STYLE-GUIDE.md** — the complete spec
2. Copy to your project
3. Import `design-tokens.css` globally
4. Merge `tailwind.config.js`
5. Follow the implementation checklist (Section 10)

## Files

| File | Purpose |
|------|---------|
| `STYLE-GUIDE.md` | **The Bible** — every page, component, state, animation |
| `design-tokens.css` | CSS custom properties |
| `tailwind.config.js` | Tailwind theme (vinyl colors, fonts, animations) |
| `constants.ts` | TypeScript — dice modes, output range, pages, colors |
| `CrateDigLogo.tsx` | React logo with spinning vinyl animation |
| `logos/*` | SVG + PNG logos |
| `icons/*` | App icons + favicons |

## Brand Summary

| Element | Value |
|---------|-------|
| **Primary** | `#F97316` (Orange 500) |
| **Background** | `#0A0A0A` (pure black) |
| **Cards** | `#111111` |
| **Display font** | Bebas Neue (logo, heroes, CTAs) |
| **Body font** | JetBrains Mono (everything else) |
| **Layout** | Single column, 600px max, no sidebar |
| **Dark mode** | ONLY mode. No light theme. |
| **Signature** | Spinning vinyl record, always visible |
